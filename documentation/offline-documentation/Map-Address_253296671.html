Created by Syed Tahmim Islam, last modified by Ahsan Habib on May 18, 2018
  1. Go to https://www.google.com/maps and search for a location and press ENTER




  2. Now click on 'SHARE' and then 'Embed a map'




  3. Now click on "COPY HTML" link to copy the HTML code for embeded map. 



  4. Now open the contact.html file and search for 'iframe'. Replace the iframe tags with the code  your have collected from google maps website.




    5. Your map location is set.
