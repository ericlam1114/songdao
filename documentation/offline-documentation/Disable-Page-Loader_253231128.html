Doc: Kryptova - Bitcoin, Cryptocurrency & ICO Landing Page Template : Disable Page Loader

Created by Syed Tahmim Islam, last modified by Ahsan Habib on Apr 26, 2018

Follow any of the processes below to disable page loader

Process: 1

If you want to customize the css file then open the main.css file from css folder and search for and .o-preloader set the selector to display:none!important; 



Note: Do not compile the sass(.scss) files if you customize the css file as it will erase the newly added css file.

Process: 2

To remove the page loader open the _base.scss  file from the sass folder and search for .o-preloader. Now set the selector to display:none!important; and then compile the file with a sass compiler.
