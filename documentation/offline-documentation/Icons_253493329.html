Created by Syed Tahmim Islam, last modified by Ahsan Habib on May 18, 2018

Change any icon of the website:

We have used a custom font named 'Cryptowallet Argon' as our main icon and 'Font Awesome' for social icons.

  1. Open the folder called 'Cryptowallet Argon'. Open demo.html file in a browser. Browse throw the page to find your required icon.
  2. Copy the name of the icon as shown the following image:



  3. Find the class name of the icon that you want to change from the html file. You can find out the name of the icon using browser console.
  4. Replace the class name with the new one.
  5. You are ready to go.
